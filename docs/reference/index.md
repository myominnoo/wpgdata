# Package index

## Core functions

Main functions for accessing Winnipeg Open Data

- [`peg_get()`](https://myominnoo.github.io/wpgdata/reference/peg_get.md)
  : Get a dataset from the Winnipeg Open Data Portal
- [`peg_query()`](https://myominnoo.github.io/wpgdata/reference/peg_query.md)
  : Query a dataset from the Winnipeg Open Data Portal
- [`peg_all()`](https://myominnoo.github.io/wpgdata/reference/peg_all.md)
  : Get all rows from a Winnipeg Open Data Portal dataset

## Metadata functions

Explore dataset structure before downloading

- [`peg_info()`](https://myominnoo.github.io/wpgdata/reference/peg_info.md)
  : Get dataset-level information from the Winnipeg Open Data Portal
- [`peg_metadata()`](https://myominnoo.github.io/wpgdata/reference/peg_metadata.md)
  : Get column names and types for a Winnipeg Open Data dataset
