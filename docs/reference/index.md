# Package index

## Discover & explore

Find datasets and understand their structure.

- [`peg_catalogue()`](https://myominnoo.github.io/wpgdata/reference/peg_catalogue.md)
  : List all datasets from the Winnipeg Open Data Portal
- [`peg_info()`](https://myominnoo.github.io/wpgdata/reference/peg_info.md)
  : Get dataset-level information from the Winnipeg Open Data Portal
- [`peg_metadata()`](https://myominnoo.github.io/wpgdata/reference/peg_metadata.md)
  : Get column names and types for a Winnipeg Open Data dataset

## Fetch data

Download rows with filtering, selection, and parallel pagination.

- [`peg_data()`](https://myominnoo.github.io/wpgdata/reference/peg_data.md)
  : Fetch data from the Winnipeg Open Data Portal
