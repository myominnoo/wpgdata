# Get a dataset from the Winnipeg Open Data Portal

Get a dataset from the Winnipeg Open Data Portal

## Usage

``` r
peg_get(dataset_id)
```

## Arguments

- dataset_id:

  A character string of the dataset ID e.g. `"d4mq-wa44"`

## Value

A tibble of the dataset

## Examples

``` r
if (FALSE) { # \dontrun{
peg_get("d4mq-wa44")
} # }
```
